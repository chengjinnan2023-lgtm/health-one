## 常见问题速查（带教员用）

| 店员说 | 可能原因 | 怎么做 |
|--------|---------|--------|
| "登录不了" | 用户名或密码输错了 | 检查大小写，确认账号已创建 |
| "搜索不到客户" | 名字打错了或客户是新建的 | 直接点新建客户 |
| "保存按钮点不了" | 有必填项没填（带 * 的） | 检查页面上的红色提示 |
| "页面白屏了" | 网络问题或系统异常 | 刷新页面（F5），不行就重登录 |
| "点错了能改吗" | 有些数据不能改 | Timeline 不能改；档案可以重新填 |
| "我不会打字" | 输入法问题 | 切换中文输入法；可以少写字 |
| "这个字段什么意思" | 术语不熟 | 参考桌上的快速参考卡 |
| "客户不等了" | 系统操作慢 | 先服务客户，事后补录；告诉观察者 |
| "我不确定填什么" | 字段不够直观 | 必填项都标了 *；不确定就跳过可选字段 |
| "我做完了吗" | 不确定是否完成 | 回总览页看 Timeline——有今天的记录就是完成了 |
