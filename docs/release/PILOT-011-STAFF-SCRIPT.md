# PILOT-011 — Staff Re-Run Briefing Script

Version     : 1.0
Audience    : 带教员 → 店员
Created     : 2026-06-30
Print       : ✅ 单页双面，朗读使用

## 开场 3 分钟

> "今天我们用 Health One 系统来服务客户。这个系统帮我们记住每位客户的健康关注、服务记录和随访计划。操作很简单——就 6 步，每步 1-2 分钟。你不需要懂电脑，只要会打字就行。如果有哪一步卡住了，告诉我——那是系统的问题，不是你。准备好了吗？我们开始。"

---

## S1：客户搜索 / 新建（1 分钟）

**讲法：**
> "第一步，找客户。如果客户来过，在这里输入名字搜索——系统会自动找。如果是新客户，点这个 '+ 新建客户' 按钮，输入名字，点 '创建客户' 就行了。"

**成功标准：**
- [ ] 客户出现在搜索结果里，或者新建成功后跳转到总览页

**卡住时怎么说：**
> "没关系，搜不到就新建。名字打错了也可以后面改。"

---

## S2：客户总览（2 分钟）

**讲法：**
> "这里是客户的总览页。上面是客户名字和状态——新客户显示 '待激活'，点这个绿色的 '激活健康元' 按钮就行。左边是健康档案和服务历史，右边是随访和最近动态。你看，系统会自动记录每次服务的动态。"

**成功标准：**
- [ ] 新客户状态变为 "已激活"
- [ ] 能看到客户的基本信息

**卡住时怎么说：**
> "如果看不到数据，别担心——新客户本来就是空的。做完服务就会有记录了。"

---

## S3：健康关注录入（2 分钟）

**讲法：**
> "现在记录客户为什么来。先选一个关注类别——肩颈、腰背、疲劳这些——点一下就行。然后在下面写上客户说了什么。比如 '长期低头工作导致颈椎不适'。其他字段可填可不填。填完点保存。"

**成功标准：**
- [ ] 类别已选中（蓝色高亮）
- [ ] 客户自述已填写
- [ ] 保存成功，跳回总览页

**卡住时怎么说：**
> "至少选一个类别，写一句话就行。不用写很多。手机号、出生年份这些不填也没关系。"

---

## S4：服务记录（2 分钟）

**讲法：**
> "接下来记录你做了什么服务。先选服务类型——健康舱、咨询、检测这些。然后写下服务内容，比如 '石墨烯舱 30 分钟'。服务前备注可写可不写。你的名字系统已经填好了。点保存。"

**成功标准：**
- [ ] 服务类型已选中（蓝色高亮）
- [ ] 服务内容已填写
- [ ] 保存成功，跳转到反馈页

**卡住时怎么说：**
> "服务类型和服务内容是必填的——其他可以不填。不用写很长，一句话就行。"

---

## S5：服务反馈（1 分钟）

**讲法：**
> "这一步记录客户的反馈。三个必填：感受——客户现在感觉怎么样？满意度——满意、一般还是不满意？回访意愿——愿意、可能还是不愿意？其他可选。填完点保存反馈。"

**成功标准：**
- [ ] 感受已填写
- [ ] 满意度已选择
- [ ] 回访意愿已选择
- [ ] 保存成功

**卡住时怎么说：**
> "就三个必填——感受写一句话，满意度和回访意愿点一下就行。很快的。如果不确定，可以跳过。"

---

## S6：随访（1 分钟）

**讲法：**
> "最后一步，创建随访。告诉系统什么时候、用什么方式联系客户。选随访方式——电话、微信、短信、到店。选一个计划时间。点创建随访。创建完你可以记录随访结果——比如 '客户感觉好了很多，下周会再来'。点标记完成。"

**成功标准：**
- [ ] 随访方式已选择
- [ ] 计划时间已设置
- [ ] 随访已创建
- [ ] （可选）随访结果已记录，标记完成

**卡住时怎么说：**
> "随访方式和时间是必填的——其他可以不填。随访不是推销，是关心客户的服务体验。"

---

## 完成后

> "完成了！你可以回总览页看——系统已经记录了今天的所有操作。看这里：健康档案有了，服务历史有了，随访也有了。以后这个客户再来，你一眼就能看到他之前的所有记录。"

**让店员自己做一遍：**
> "现在换你来——用这个系统再做一遍。我在旁边看，不说话。有卡住的就问我。"
