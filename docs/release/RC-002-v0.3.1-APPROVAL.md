## 2. Status: Release Candidate

v0.3.1 is formally marked as **Release Candidate**.

This designation means:
- The codebase is feature-complete for the manual service loop
- All P0 features (F1–F9) are implemented and verified
- All QA reviews are approved (10/10)
- No known blocking defects exist (6 bugs found, 6 fixed)
- The codebase is ready for server-side deployment preparation

This designation does **NOT** mean:
- The software is deployed to a store server
- Real customer data can be ingested
- The manual loop has been validated with real staff
- Pilot status is approved (requires separate Founder approval per FD-006 §4.3)
