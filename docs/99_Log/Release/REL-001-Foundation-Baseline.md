## 4. Baseline Decisions Confirmed

As of this release, the following decisions are Baseline:

1. The project working name is Health One.
2. The Chinese concept name is 健康元.
3. Health One is a long-term health management ecosystem.
4. Health One is not merely a mini-program, SaaS, CRM, device booking system, marketplace, or AI chatbot.
5. Git is the formal source of truth.
6. The repository is the formal project knowledge base.
7. Formal project decisions require review and release.
8. Baseline documents are protected.
9. AI collaborators must follow GOV-001, GOV-002, and GOV-003.
10. Claude Code currently acts as Release Engineer.
11. The Founder holds final approval authority.
12. The Chief Architect protects structure and long-term coherence.
13. The current MVP should focus on the shortest complete health value loop.
14. The customer should be treated as the owner of their 健康元 and health memory.
15. Trust, memory, real service, and store operating capability are core assets.
16. Premature complexity should be avoided before the core value loop is proven.
