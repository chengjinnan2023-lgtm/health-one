# PITCH-001 Summary

PITCH-001｜Health One 门店联盟合作方案 V1.1 已正式入库，作为首份面向联盟合作门店的品牌合作资料。

## Key Content

- **核心主张**：不是给门店增加一个项目，而是帮助门店建立未来十年的客户经营能力。
- **核心定位**：Health One 是平台，健康元是核心资产。
- **当前入口**：纯石墨烯舱为当前第一入口，不是唯一入口。
- **样板政策**：10 家样板门店，已有 4 家，继续寻找 6 家；1 万元押金，1000 元进出场安装费，20 元/次预充值，免费训练门店专属 AI。
- **合规边界**：不做医疗治疗承诺，表达聚焦舒缓、恢复、状态管理。
- **落地路径**：90 天样板门店落地路径｜一年陪跑。
