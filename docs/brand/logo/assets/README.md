# Health One Temporary Logo Assets

## Current Status

当前目录用于承接 Temporary Logo v0.1 最小可用资产集。

---

## Important Rule

当前仓库中不允许伪造正式 Logo 文件。

只有在真实导出完成后，才能将 SVG / PNG / PDF 放入对应目录。
