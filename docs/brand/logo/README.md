## Asset Folder Structure

```
assets/
  primary/        — Primary Logo (symbol + wordmark + 健康元)
  horizontal/     — Horizontal Logo
  stacked/        — Stacked Logo (symbol above, text below)
  icon/           — Icon Mark (symbol only)
  wordmark/       — Text-only Logo (Health One / Health One｜健康元)
  monochrome/     — Black / White / Deep Green versions
  preview/        — Mockup previews for website, PPT, FAQ, App, storefront
```
